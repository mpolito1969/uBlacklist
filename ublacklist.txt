# Questa blacklist serve per evitare che nei
# risultati delle ricerche vengano mostrati siti che
# subordinano l'accesso ai contenuti al pagamento
# di un abbonamento o all'accettazione di cookie.
# 
# Per accedere comunque ad uno di questi siti, anche
# dopo aver importato la blacklist (ad esempio nel
# caso di persone abbonate ad uno o più dei siti
# presenti), è sufficiente identificare la regola che
# lo blocca (es. "*://*.ansa.it/*"), entrare nella
# pagina di configurazione dell'add-on, e inserire
# tale regola nella text box "General" preceduta da
# "@" (es. "@*://*.ansa.it/*")
# 
# La lista viene aggiornata man mano che trovo siti
# simili o che mi vengono segnalati.
#
# Contatti: @max@poliverso.org
#
